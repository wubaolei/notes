Java 与 javascript区别：1.java 基于类 class 的面向对象, javascript 基于方法 function 面向对象且不仅仅面向对象
			2.java强类型 显式	JavaScript弱类型 隐式
			3.java没有函数类型	JavaScript有函数 并且可以赋值给一个变量
			4.


java注意点：1.java 到 class的编译期间有错误时不会编译成class文件 （ 编译期间 语法错误 )
	    2.运行class文件时 可能会有错误	（ 运行期间错误 ）
	    3.只要首字母大写 就是类class    类名 必须大写
	    4.一个类中只能有一个 main 函数 一个系统只能有一个入口
	    5.跨类访问，过程不变
	    6.静态变量是属于整个类，不存在于某个方法内
	    7.访问修饰符private 只能修饰 类 方法 全局变量，不能修饰局部变量
	    8.创建java类要建立包
	      包名小写
	      包名的命名：com.缩写名字.功能名
	    9.所有的类，都在包里面
	    10.同一个包中的类不需要导入，但是不同包中的类，需要导入才能使用
	    11.java中的所有类都是 直接 或 间接继承object类(顶级类)
	    12.


Java特点：1.跨平台的语言  （ 有很多 jdk 文件 ）

Java面向对象：1.封装：
		    1.1.对类中的属性( 数据 )进行私有
		    1.2.提高数据访问安全性
		    1.3.通过方法数据访问，在方法内控制数据安全
	      2.继承 父类和子类 作用：减少代码的冗余
			父类可以抽取子类中相同属性，方法的能力并给子类继承
			子类具有自己特色的属性和方法
		    
	      3.多态 多种表现形式 具体有两种：
				3.1.重载 在一个类中，相同的方法，不同的实现结果，也就是变现结果
					具体条件：
						3.1.1.同一个类
						3.1.2.相同的方法名
						3.1.3.不同的参数列表（ 个数 参数类型 ）<-- 决定调用哪个方法重载
					注意：
						3.1.1.与返回值无关
						3.1.2.静态方法也是方法重载
						3.1.3.方法的重载叫做，编译时多态
				3.2.重写
	      4.抽象
